# Zenith UI and product spec

- I want a beautiful UI the beige / cream backgrounds, pastel colors and luxury
  fonts
- On the home screen I want to show a circular dial for the break down of the
  days activites what was done / accomplished / completed
- I want a daily wrapped which shows a break down of the skills leveled up and
  the parts of the personaltiy improved.
- I want to have different archetypes which is assigned based on the goals and
  habits the user has set / performs
- I want to assign generic habits based on problems they hilight in their life
  and also i want to assign specific habits based on their specific goals and
  desires
- I want a voice log where we can tap the button record their voice and
  determine the activites and time spent on said activites + the elements of the
  personality that it improves
- Schedeule reminders which send notifications to complete a specific task
- Daily wrapped, weekly wrapped, monthly wrapped and yearly wrapped with slick
  animations and break downs by the AI + beautiful graphcis and visuals -
  biggest selling point / tiktokable moment
- Overall UI design principles - sleek, luxury, beautiful, minimalist
- The onboarding process higlights the problems in their life, habits can be
  habits of abstience eg: no porn / social media, habits of performance eg:
  reading, writing coding, journalling etc.
- Information based onboarding slides should be:
  - animation
  - text header
  - subtext
- Onboarding slides for user to answer questions should be:
  - typewriter text with haptic feedback to type out the question
  - list of otpions to choose from
